# SlitherBet

SlitherBet is a playable React/Vite demo for a casino-integrated snake arena concept. The current build focuses on the game feel and product surface: lobby tiers, buy-ins, rake-adjusted prize pools, a simulated multiplayer field, money-dot collection, eliminations, leaderboards, final-six payouts, and responsive UI.

## Run locally

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:5173`.

## Controls

- Move with the pointer over the arena.
- Hold the Boost button or press Space to boost.
- Eat glowing dots to grow and increase your claim.
- Avoid other snakes, survive to the final six, and play for the payout ladder.

## Current scope

This is a front-end prototype with simulated opponents. Real-money wallet handling, casino operator integrations, compliance, accounts, matchmaking servers, and production multiplayer are intentionally left for later phases.
